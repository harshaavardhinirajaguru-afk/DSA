City Path Finder

 Overview
This project implements a City Path Finder using Dijkstra's Algorithm in C++. Cities are represented as nodes and roads as weighted edges. The program finds the shortest route and total distance between two cities.

 Features
- Represent cities as graph nodes
- Add roads with distances
- Find the shortest path between two cities
- Display the shortest route and total distance

 Concepts Used
- Graph
- Dijkstra's Algorithm
- Priority Queue
- Vectors
