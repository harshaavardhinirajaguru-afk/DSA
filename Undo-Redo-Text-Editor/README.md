Undo-Redo Text Editor Simulation

Description
This project simulates the Undo and Redo functionality of a text editor using Stack Data Structure in C++.

Features
- Type text
- Undo last action
- Redo last undone action
- Display current text

 Data Structure Used
- Stack (STL stack)

 Concepts Used
- Stack
- Functions
- Strings
- Menu Driven Program

 How to Run

Compile the program

g++ main.cpp -o editor

Run

./editor
